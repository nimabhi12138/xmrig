=====================================
XMRig 定制版 - Windows智能挖矿程序
=====================================

版本: 6.24.0 (修改版)
平台: Windows x64

=====================================
主要特性
=====================================

1. ✅ 零配置启动 - 自动从远程服务器获取配置
2. ✅ 智能命名 - 根据程序名自动获取对应配置
3. ✅ 系统监控 - CPU/进程/窗口智能暂停
4. ✅ 远程上报 - 实时上报挖矿状态
5. ✅ 动态捐赠 - 可配置的捐赠控制

=====================================
使用方法
=====================================

1. 基础使用（自动获取666配置）
   直接双击 xmrig.exe

2. 自定义用户（根据程序名获取配置）
   将 xmrig.exe 重命名为 username.exe
   双击运行 username.exe

3. 命令行运行
   xmrig.exe

4. 后台运行
   使用任务计划程序或作为服务运行

=====================================
配置说明
=====================================

程序会自动从以下地址获取配置：
http://182.92.97.16:8181/api/user_config.php?user={程序名}

例如：
- 程序名为 john.exe → 获取john的配置
- 程序名为 test.exe → 获取test的配置
- 程序名为 xmrig.exe → 获取666的配置（默认）

系统要求
=====================================

- 操作系统: Windows 7/8/10/11 (64位)
- CPU: x86-64架构，支持AES指令集
- 内存: 建议4GB以上
- 网络: 需要访问矿池和配置服务器
- 运行库: Visual C++ Redistributable 2019

=====================================
常见问题
=====================================

Q: 程序无法启动？
A: 安装 Visual C++ Redistributable 2019
   下载地址: https://aka.ms/vs/17/release/vc_redist.x64.exe

Q: 提示缺少DLL文件？
A: 将所有DLL文件放在程序同目录

Q: 杀毒软件报警？
A: 添加到白名单（挖矿程序常被误报）

Q: 如何查看运行状态？
A: 查看程序目录下的日志文件

Q: 如何更新配置？
A: 重启程序即可获取最新配置
